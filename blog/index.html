I haven't updated my website for a long time (6 months) because I couldn't find time. Now, it's semester holiday and I decided to update with the new apps I developed. Also, I updated the Story Sea app with reading modes and new fonts. There is a problem with the page mode, if I try to change the font or the size, app crashes. I guess it's because of LayoutInflator. So, I removed this feature from page mode. Lastly, I started an Unity course. I'm trying to learn develop games (I mean real games, not like GMCB or Pixel Memory). I know nobody reads this blog but this is something like a diary, my developer diary.
